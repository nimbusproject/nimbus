Requirements
============
CherryPy_

.. _CherryPy: http://www.cherrypy.org/

Installation
============

1. Put ``django_cpserver`` on your ``PYTHONPATH``.
2. Add ``django_cpserver`` to your ``INSTALLED_APPS``.

Usage
=====

Run ``./manage.py runcpserver help`` from within your project directory
